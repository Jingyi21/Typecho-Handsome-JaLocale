## 使用方法

**注意：使用本设置需要拥有 Handsome 主题**

1. 打开 GitHub 项目页面
2. Clone 或 Download ZIP 本项目 ~~如果有 Star 的话我会更高兴的~~
3. 解压压缩包，得到 jp.php 文件
4. 使用 SSH 或 FTP 或管理面板将 jp.php 上传至 `/usr/themes/handsome/usr/lang`
5. 进入 Handsome 后台管理 - 主题增强管理 - 设置语言，选择“日本語”
6. 保存设置后刷新主页
7. Enjoy
